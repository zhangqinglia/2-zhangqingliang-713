# 高级筛选
在Excel表格中选出一个空白位置，在上面输入“数学”，在数学下面输入“>80”，同时在“数学”的同一排右侧输入“英语”，在英语下面空一格输入“>80”，然后点击上方工具栏中的数据，找到筛选旁边的高级，点开后在列表区域中选中自己要选择的包含数学、英语成绩的区域，在条件区域中输入自己刚才所输入的条件的区域，然后点击确定，就可从中挑选出‘英语成绩高于80分或数学成绩高于80分的学生’。
